# Compose to Kubernetes Workshop

A hands-on workshop for developers transitioning from Docker Compose to Kubernetes.

**[Workshop Documentation & Labs](https://rfaes-com.github.io/compose-to-kubernetes/)** · **[Slides Part 1](https://rfaes-com.github.io/compose-to-kubernetes/slides/part-1.html)** · **[Slides Part 2](https://rfaes-com.github.io/compose-to-kubernetes/slides/part-2.html)**

## Overview

**Duration:** 8 hours total (two 4-hour sessions)
**Format:** Instructor-led with hands-on labs
**Prerequisites:** Docker Compose experience, basic terminal usage

## Getting Started

1. Clone this repository and open it in VS Code *(Windows users: clone into a WSL volume for better performance)*
2. Click **Reopen in Container** when prompted
3. All tools are pre-installed and ready

## Workshop Content

### [Part 1: Kubernetes Fundamentals](docs/part-1/)

Pods, Deployments, Services, ConfigMaps, Secrets, Storage, Namespaces, and a final 3-tier application lab.

### [Part 2: Advanced Operations](docs/part-2/)

Ingress, Helm, GitOps with Flux, Monitoring, Advanced Deployment Strategies, Autoscaling, Security, and Multi-Cluster.
